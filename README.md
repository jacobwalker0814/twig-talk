# Twig Quick Talk

A short presentation for [UpFront Wichita](upfront) about the PHP Templating language [Twig][twig].

## The Presentation
View the presentation with
~~~bash
cd presentation
php -S localhost:8000
~~~

## The Demo App
View the demo app with
~~~bash
cd app
php -S localhost:8001
~~~
